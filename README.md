# Financial KPI Dashboard

## 📊 Overview
This Power BI dashboard provides insights into key financial metrics:
- Revenue Trend vs Target
- Profit Margin by Month
- Cash Flow Analysis
- Interactive Slicers for filtering by Month

The goal is to help visualize performance, identify strong/weak months, and track financial health at a glance.

---

## 🎨 Features
- Custom 8-color theme for a professional and attractive design
- KPI cards for quick summary of Revenue, Profit Margin, and Cash Flow
- Clustered Column Chart with conditional formatting (Profit Margin by Month)
- Line Chart with constant target line (Revenue vs Target)
- Interactive slicers for filtering by Month

---

## 🛠️ How to Use
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop**.
3. Explore the dashboard by applying slicers and reviewing KPIs.
